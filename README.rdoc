== Планировщик личных задач

* добавление, удаление, редактирование задач
* установка приоритета
* возможность разбиения по категориям (работа, учеба)
* возможность отметить задачу как выполненную/не выполненную
* установка зависимостей (например, задача № 1 должна быть выполнена после задачи № 2) и т.д.
